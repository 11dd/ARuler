# ARuler
Mesure distance using apple ARKit

## 预览
  <img src="./Design/preview.jpg" width="100%"/>
  
## 安装
因为ARKit使用限制，设备要求为A9以上，系统最低要求为iOS11

## 问题
ARKit目前只能识别水平的平面，所以只能测量水平面的长度，不能测量垂直平面上的长度，不知道视频里的是怎么实现的，有知道的小伙伴可以一起交流一下
